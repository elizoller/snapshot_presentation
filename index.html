Digital Repository Service DRS

Bulk Loaders - Solution

  1. IPTC Photograph Loader
  2. Multipage Loader
  3. XML Loader - with metadata and files, with metadata only
  4. Spreadsheet Loader - with metadata and files, with metadata only

IPTC Photograph Loader

  • University Marketing and Marketing departments in various schools and colleges
  • Zip file of JPGs
  • Metadata extracted from IPTC header and translated into MODS
  • ~40,000 photos to date

Multipage

  • Library staff - originally designed around a specific library collection
  • Zip file of images and their associated MODS XML metadata
  • Loads sequential images for objects like postcards and page images
  • ~1,500 files to date

XML

  • Archives and Special Collections and Library staff
  • ~4,000 files
  • Zip file MODS XML (and files for new objects)
  • Metadata + Files: Loads new files with their associated metadata
  • Metadata Only: Replaces existing MODS XML with provided file

Spreadsheet

  • Archives and Special Collections and Library staff on behalf of other projects
  • ~10,000 files
  • Zip file containing an excel file (and files for new objects)
  • Metadata + Files: Loads new files with their associated metadata by transforming spreadsheet metadata into MODS XML
  • Metadata Only: Replaces existing MODS XML with spreadsheet metadata transformed into MODS XML
